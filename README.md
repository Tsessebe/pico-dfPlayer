# pico-dfPlayerMini

This is a simple library for DFPlayer Mini using the Raspberry Pico.
Currently it does not support reading the responses from the unit.
More features will be added in the future.

## Requirements:
- DFPlayer Mini
- Raspberry Pico SDK 1.4

## References:
